centos6.5笔记本wifi网络配置

iiitoo
2014-01-01

参考：转载：http://u50.cn/blog/centos-6%E4%B8%AD%E9%85%8D%E7%BD%AE-broadcom-%E6%97%A0%E7%BA%BF%E7%BD%91%E5%8D%A1#comment-60259

有一台老的 dell 1400 笔记本，速度已经跟不上时代了，闲置下来。

拿它来做一个服务器，用于自己网站建设测试。

前面网卡驱动配置与参考文件不一样，再说，每个人电脑的网卡不同，就没有写出。无线上网，全靠下面引用的文章。

现在网卡配置好了，下一个任务是连接 WIFI 并在开机时启动

参考文档：http://www.pantz.org/software/wpa_supplicant/wirelesswpa2andlinux.html

 

1. 找 wifi 热点

sudo iwlist scan

2. 生成热点的配置（wpa2加密）

sudo wpa_passphrase <wifi热点名称> <wpa密码>

会生成这样格式的配置代码：

network={ 

        ssid="SSID_xxx" 

        #psk="your_pass_here" 

        psk=eb900177d2234487397b54f1fc1d6efc7f30e43b988ad21488ae3eb701a1g3b0 

}

将这段代码添加到 /etc/wpa_supplicant/wpa_supplicant.conf 文件中：

ctrl_interface=/var/run/wpa_supplicant 

ctrl_interface_group=wheel

network={ 

        ssid="SSID_xxx" 

        #psk="your_pass_here" 

        psk=eb900177d2234487397b54f1fc1d6efc7f30e43b988ad21488ae3eb701a1g3b0 

}

配置就完成了，下面添加启动代码

3. 在 /etc/rc.local 中添加相应的 wifi 启动代码：

pkill wpa_supplicant 

pkill dhclient

ifconfig wlan0 down 

ifconfig wlan0 up 

iwconfig wlan0 essid "SSID_xxx" 

wpa_supplicant -B -Dwext -iwlan0 -c/etc/wpa_supplicant/wpa_supplicant.conf

dhclient wlan0

现在重启系统，无线网络就会配置好了
