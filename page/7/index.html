平衡

人的生存环境里，总是要寻求一种平衡。不管是无知做为主导还是理性作为主导，最终归为平衡。
争斗或者对抗，是某些人对现实环境的不满对这个环境的一个作用，不管这个人是代表着恶还是善，在对抗的僵持过程中，他能够在时间上一直坚持下去。他的欲求最终就会转换成为一种现实。就像一个人打在自己身上的耳钉一样，一开始身体是排斥的，只要时间够久，他就会成为身体的一部分。身体不会因为有这样一点小的东西去对耳朵作出更大的惩罚。人在这个仿效生物机能的环境机制里也同样会发生这样一种情况。

我希望大学时就能明白的6件事

这两天读一本很简短的书叫做《Steal Like An Artist》，摘录了一些段落，还挺多共鸣的，总结下来也差不多是这几年来我学习到的最多的几个东西。列举如下：

1. 别把上过大学就当成受了教育，把空余的时间多用来读书。

School is one thing, education is another. The two don’t always overlap. Whether you are in school or not, it’s always your job to get yourself an education.

Always be reading, go to library, there’s magic in being surrounded by books. Get lost in the stacks. Read bibliographies. It’s not the book you start with, it’s the book that book leads you to.

大学和教育是两码事，可惜我们的教育让大多数人在大学结束后学习的过程就停止了，从某个高度上来说，学习是一种人生态度，是一辈子的事情。我觉得最简单的改变就是让你的思想吸纳精华，上千年来人类思维的精华都浓缩在书里，学习如何去阅读，大量阅读，当你有时间无所事事的时候，读书；把手机里的游戏卸载了，读书；等人等车排队坐地铁的时候，读书；这在我看来比什么都重要。

我相信学习的力量，当你迷茫的时候，不知所措的时候，把精力直接放在学一些新东西，读一本好书上。

2. 旅行，去看世界

Your brain gets too comfortable in your everyday surroundings. You need to make it uncomfortable. You need to spend some time in another land, among people that do things differently than you. Travel makes the world look new, and when the world looks new, our brains work harder.

In the meantime, if you are not into the world you live in, you can build your own world around you.

无需多言，你只需要踏出第一步。如果你对周遭不满意，想办法创造一个适合你的环境。

现在对这个话题没有那么大兴趣了，主要觉得旅行给你建立的实际上是一个思维模式和看世界的角度，你有了这套武器你的人生有趣程度会直线上升，这样即便在一个你熟悉的城市落地申根，旅行带给你的触角还是会让你的每一天都不一样。

3. 我们只有在试错的过程中才能发现对的

In my experience, it’s the act of making things and doing our work that we figure out who we are.

What are shaped and fashioned by what we love. -Goethe

去冒险，去体验，去尝试，别相信任何人告诉你的职业生涯规划，如果你想要一个丰富多彩的人生，唯一的办法就是让它大起大落，折腾不惜，虽然有时让你看起来很落魄，但结果往往你会更快乐，也更成功。

4. 学习创业家的精神，努力做事，乐于分享

Do good work and share with people, it’s a two step process, step one, “do good work” is incredibly hard. There are no shortcut. Make stuff every day. Know you’re going to suck for a while. Fail. Get better. Step two, “share it with people”, was really hard until about ten years ago or so. Now it’s very simple: put your stuff on the Internet.

Draw the art you want to see, start the business you want to run, play the music you want to hear, write the book you want to read, build the products you want to use, do the work you want to see done.

You have to dress for the job you want, not the job you have, and you have to start doing the work you want to be doing.

6. 辞职需谨慎，明白自由职业并没有想像中那么好

A day job gives you money, a connection to the world, and a routine, freedom from financial stress also means freedom in your art.

The worst thing a day job does is take time away from you, but it makes up for that by giving you a daily routine in which you can schedule a regular time for your creative pursuits. Establishing and keeping a routine can be even more important than having a lot of time.

本真

怎么做自己，应该做自己本真还是做自己能做的？自己的本真又是什么？假如，我现在喜欢做网站，一些技术方面的事情，但是这就是我的本真吗？
我在此推测一下，是不是应该不要太注重与人交往上的情感上的本真，而是，要去找到自己爱做的那件事情的本真？假设，我喜欢睡觉，吃饭，是不是应该去学会怎么样才能各种睡，最终成为一个很有一套的睡觉方式？吃饭也是，是不是，应该成为美食家？我们再找本真，就是自己最真实最想做的一面，自己本应该做的一面。那么，人又是有想象力，脑子里的想像永远都不能和现实中一样，我们又是不是对此自暴自弃，不作为。这样肯定不行，只有一个方法，在保证不危害身体的同时尽量向着哪个方向靠近。

写WordPress博客的三四年

我是个蠢人，还有一些强迫症。一直想用WordPress做一个网站。自己对PHP不是很了解，js也是一知半解。然后用别人的主题，总是不满意不满意，要么修改主题，要么扩展插件。技术倒是没学到一点，博客也全写的是“测试，测试”，但是各种无边的欲求越来越多。使得自己压力又很大，做的事情进度又很慢。
现在终于解脱了，不是因为我会做了，而是知道了一点点经验。人的贪欲越多，他的压力自然越多。他要去完他所希望的事情，但是又太多样了。总是把这个做百分之六十，就去做哪个，来来回回，事事无成。
作为WordPress，他就是一个cms，如果自己想写点博客，那就用它搭建起来，开始写就好了。不要把太多精力耗在主题，插件啊修改了又学习，到了最后还是不会。我就克制去做这些无用的事情，就用现在这样一个主题就很好。然后，写写博客。就很OK。就行了，
如果说，做开发，就不要在人家开发好的东西上开发什么了。我们还是需要一步一个脚印从最低处走向这个台阶才是最好的，最捷径的。
借此，我就大声的说，“我的网站做好了，我开始些博客了！哈哈哈”

MWeb - 专业的 Markdown 写作、记笔记、静态博客生成软件

专业的 Markdown 写作支持

  • 极简 UI、Dark Mode、漂亮的 Markdown 语法高亮、列表缩进优化，提供 5 种主题选择。
  • 除了支持基本的 Markdown 语法外，还支持大量 Markdown 扩展语法：Table、TOC、MathJax、Fenced code block、任务列表（Task lists）、顺序图和流程图、Footnote 等。
  • 支持 Typewriter Scrolling（打机滚动模式）View - Typewriter Scrolling。
  • 支持发布和更新到：Wordrpess 博客、支持 Metaweblog API 的博客服务、Wordpress.com、Evernote 和印象笔记、Blogger、Tumblr。请在 Preferences - Publishing 增加发布服务，然后点击软件右上角的分享按钮即可看到所增加的发布服务。
  • 支持即时预览并提供 6 种预览主题，其中二种和静态博客主题相对应，也就是说您在写博客时可以即时预览大概效果！所有主题效果都支持导出为 HTML、PDF。快捷键 CMD + R 或 CMD + 4 打开即时预览窗口。
  • 编辑器和实时预览都支持大纲视图，长文档时跳转非常方便。

设计为两种模式

  • 外部文档模式：用于新建、打开和编辑外部 Markdown 文档。也支持引入外部文件夹到 MWeb 中管理。
  • 文档库模式：用分类树管理文档，可以把文档设为多个分类，用于记笔记和静态网站生成。

CMD + E 或使用菜单 View - Open External 可打开外部文档模式。

CMD + L 或使用菜单 View - Open Library 可打开文档库。

文档库模式和外部模式都支持全文搜寻（Full Text Search），都可以用拖放或粘贴插入图片并直接显示。CMD + V 粘贴为JPG格式，CMD + Shift + V 粘贴为PNG透明格式。

外部模式引入 Octpress、Jekyll 等静态博客的文件夹后也支持拖放或粘贴插入图片和实时预览，详细请参考：引入文件夹到 MWeb 中管理，支持 Octpress、Jekyll 等静态博客拖拽插入图片和实时预览

文档库模式用于记笔记

文档库模式使用分类树组织和管理文档，支持拖放或粘贴插入图片并直接显示，插入非图片则会生成连结。

支持把 Markdown 或文本文档导入到文档库，也支持把整个分类或者文档（可选多个）导出为 HTML、PDF、Markdown。

更详细的信息请看：MWeb 文档库模式详细说明

文档库模式用于静态博客生成

一键把分类生成静态博客，目前可选二个主题，支持自定主题。只要填入 Disqus、多说提供的代码即可以为博客增加评论功能。可勾选让网站支持 MathJax 和顺序图、流程图。

更详细的信息请看：MWeb 生成静态博客详细说明、静态博客功能增强

bootstrap

什么是 Bootstrap？

Bootstrap 是一个用于快速开发 Web 应用程序和网站的前端框架。Bootstrap 是基于 HTML、CSS、JAVASCRIPT 的。

历史

Bootstrap 是由 Twitter 的 Mark Otto 和 Jacob Thornton 开发的。Bootstrap 是 2011 年八月在 GitHub 上发布的开源产品。

 

为什么使用 Bootstrap？

  • 移动设备优先：自 Bootstrap 3 起，框架包含了贯穿于整个库的移动设备优先的样式。
  • 浏览器支持：所有的主流浏览器都支持 Bootstrap。


  • 容易上手：只要您具备 HTML 和 CSS 的基础知识，您就可以开始学习 Bootstrap。
  • 响应式设计：Bootstrap 的响应式 CSS 能够自适应于台式机、平板电脑和手机。更多有关响应式设计的内容详见 Bootstrap 响应式设计。

  • 它为开发人员创建接口提供了一个简洁统一的解决方案。
  • 它包含了功能强大的内置组件，易于定制。
  • 它还提供了基于 Web 的定制。
  • 它是开源的。

Bootstrap 包的内容

  • 基本结构：Bootstrap 提供了一个带有网格系统、链接样式、背景的基本结构。这将在 Bootstrap 基本结构 部分详细讲解。
  • CSS：Bootstrap 自带以下特性：全局的 CSS 设置、定义基本的 HTML 元素样式、可扩展的 class，以及一个先进的网格系统。这将在 Bootstrap CSS 部分详细讲解。
  • 组件：Bootstrap 包含了十几个可重用的组件，用于创建图像、下拉菜单、导航、警告框、弹出框等等。这将在 布局组件 部分详细讲解。
  • JavaScript 插件：Bootstrap 包含了十几个自定义的 jQuery 插件。您可以直接包含所有的插件，也可以逐个包含这些插件。这将在 Bootstrap 插件 部分详细讲解。
  • 定制：您可以定制 Bootstrap 的组件、LESS 变量和 jQuery 插件来得到您自己的版本。
