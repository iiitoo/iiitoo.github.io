大气魄

一个人的大气魄，就是随风飘去的树叶，并不能让整颗树死去。这就是一个人应该有的大气魄，树之长命百年千年，人之智慧无穷尽。怎能一两个小心思就能让人衰竭耗尽。
让其随风飘去，只是点点的凄美。显入眼影！

补忆

大傻与小妞

这是一个梦幻的地方，一个没有被繁华与喧嚣污染的地方。安静时，只有知了和小鸟，或许还多了我们几个小孩玩闹的声音。那是发给各自母亲的安全信号。这也是一个与世间隔开的地方，但隔开的并不明显。是一个天堂，我们的天堂——一个被世间可以看到的天堂，是世人都想进去的天堂。这里有世间有的，也有世间没有的，就这样完美的地方。
就在这样的一个地方，大傻和小妞在这里成长。大傻的玩伴是小妞，小妞的玩伴可多了。小气的大傻虽然不情愿，但是小妞的生气是大傻不能承受的。也只好和小妞的那些玩伴们一起和平共处，只是在大傻的心里他们只是让小妞高兴的玩具。说别人是玩具，其实，真实的大傻也同样是小妞的玩具，大傻喜欢让自己作为玩具让小妞去玩。在那个时候，没有女神这个说法，如果有，小妞就是大傻的女神。大傻爱战争的时候，小妞总能让大傻明白和平，大傻不爱看书，大傻的书全被小妞看光了。大傻喜欢站在小妞旁边看她看书。大爱吃好吃的，但是小妞总是让给大傻吃，大傻就脸红了。不过大傻总是忧心忡忡的想些什么，也时不时的和别的孩子意见不同，有小妞，大傻就不敢啃声。
小妞是一群孩子中的头头，是大傻他们的统帅。小妞是一个很民主的统帅，她总听大傻的一些建议，也会听别的孩子的建议。她总能帮这一群孩子找到快乐，她能用她特有的魅力，软性的统治这一群孩子，她比大傻小几个月，她很快乐，她带着别的孩子也很快乐。大傻总是从心底里带着一些叛逆，他时常找一些麻烦，然后，小妞就会解决掉。从很小很小的时候，小妞带着孩子们玩大傻的玩具，当小妞玩大傻的玩具的时候，也是大傻最高兴，最引以为豪的时刻。她拿着玩具，在用她各种想象力给玩具增添着孩子们快乐的魔力。稍微大点的时候，大傻从学校里别的学生跟前学着抽烟。他还想让小妞也抽，他旁敲侧击的试了两三次，然后以为小妞也会跟着他抽烟。小妞拿到烟便用她的好奇心玩弄了几下，就了事。小妞就是这样，总是无拘无束。没有他爸妈给他的压力，学习成绩总是很好。大傻总是高兴并羡慕着。
大傻是小妞的忠实粉丝，只要小妞来大傻家玩。大傻也总能得到一个很大的礼物，就是大傻的妈妈让大傻去和她玩。大傻，表面总是傻乎乎的，但是心里总是好像存着各种忧愁的心思。大傻的胆子很小，大傻是个左撇子，所以大傻总是能想出各种和别的孩子不同的想法。大傻生长在甜言蜜语的蜜罐中，他总会被别人骗。只有小妞从来也不夸大傻，但是大傻还是那么喜欢和小妞玩。大傻生长在一个很大很大的院子中，也只有大傻可以在那么大的，那么平的水泥院中玩耍。大傻在没上学的时候，总是那么孤独，因为没有别的孩子可以到大傻的院子中玩耍。大傻从能看到别人的时候，他的父亲就有了好多好多，有德高望重的老“父亲”，也有潮气蓬勃的年轻小伙子“父亲”，有不三不四的坏“父亲”，有油嘴滑舌的油“父亲”，又邋遢的“父亲”，有干净利落的“父亲”，大傻就有这么多的父亲。大傻也总是被这些父亲们影响着，但是大傻，也与这些父亲们时不时的离别，他会走，走了也许就再也不会回来了。当他们走的时候，大傻有些快乐中夹杂着一些忧心。大傻不懂，大傻总是不能理解，但是大傻就不去理解了。大傻还有一个水池，有一大片田地，有一个小山坡，那里有各种不同的动物和植物，有三个花园，但是花园中的花总是月季花。大傻有一个乡的不属于大傻的粮仓和粮食。他的父亲们总是给他打井，在最干旱的时候，井里的水还是那么多。天堂之外的人说，那是一块宝地，鬼魂都不能进去。所以，大傻虽然胆小，但大傻从来没有遇到鬼。大傻有这么多，这么多的好东西，但是大傻总是那么孤独，他站在他的大铁门后面总想让天堂之外的小伙伴来玩，但是他们确实进不了。
只有小妞，她可以进来。到大傻的院子里玩耍，无拘无束。大傻自然很高兴，因为大傻是那么的孤独。大傻不敢出那个天堂般的院子，因为大傻有好多好多的东西，他怕丢了。只有小妞才能引大傻出去。小妞是大傻的领袖，也总是来的次数很少很少。在很长很长的一段时间里，也只有小妞可以把大傻吸引到天堂之外的地方玩耍。他们一起开飞机，一起当老师，当学生，过家家。用那么美妙的想象力来编剧那么美妙的大人生活。他们看到大傻院子里看门人是高兴的时候，他们总跑到后院的树林里，一会儿摘杏子，一会儿摘毛桃，一会儿看养着的猪的丑态，还时不时的去发现一些别的动物。不过大傻一个的时候，他时常玩的就是蚂蚁了。当他们累了的时候，他们总想诠释一些从大人身上学到的名词。七嘴八舌的争吵，但是，小妞的诠释，永远都是最正确的诠释。大傻不知道给小妞他最心爱的东西，小妞也无所谓，因为当她想要的时候，也从不去见外，直接玩就是了。大傻很高兴。
确实，大傻就是很高兴！
大傻和小妞就这样，没有什么惊天的情感。当小妞转学的时候，大傻已经2、3年级了，大傻知道了害羞了。小妞要转走了，大傻只是忧愁了半天，就继续他自己的事情去了。只是，也可能在很长一段时间，刻在了大傻的心里，当大傻更大一点的时候，大傻以为小妞就是他喜欢的女朋友。也在很长一段时间里，大傻的女朋友原型，就是小妞。

git颜色配置

我的git全局配置文件在
~/.gitconfig
git config --global color.diff auto
git config --global color.status auto
git config --global color.branch auto
上面为我的全局配置文件
而又一些当前文件的配置文件需要单独的配置，文件在当前文件下的.git/config文件中
[user] name = *** email = ***@***.*** [core] repositoryformatversion = 0 filemode = true bare = false logallrefupdates = true [color] diff = auto status = auto branch = auto

小思

技能

在这个世界中，学习大部分时间都在学习许多许多的技能。尤其是在这个科技疯了一样的年代，而教育总觉得是落后的年代。笨人在不停的努力学习，学习各种各样的技能。但总是用不了，在任何地方，任何时间。痛恨，直至自卑，最后放弃。因为失望，因为承认自己的“愚蠢”。
冥冥中感觉到，不是技能的问题，只是需要一种方式去证明自己。不是从这个世界获取了多少经验的事情，因为我们还在承认我们的时间不过百年。短短的时间，不会像上帝一样，懂得所有。从此取其一勺，便是享受这世间。我要两勺，三勺，那一锅，首先是撑，然后便是撑死。经验也是如此，不是什么“技多不压身”谬谈！
技能的学习，若我不是天才，那么如果只停留在脑海中，没有把它应用出来。学习那到底能成为什么？可能就是忘记了，然后去学习新的东西。

主流的毒

主流的毒，让您失去了自己，这个失去又让您的脸上增添了皱纹，因为烦恼。主流是大众的公知，让大家去学。学会了，您便得到你本来没有的东西。这是惊喜，但这本不是您想要的，也就让您快乐那么一会儿。然后，用主流的公知，去得到您下一个惊喜，下一个不想要的东西，下一条皱纹。老天，一开始给我们总是最适合自己的，因为没有惊喜，你丢掉了它，然后失去了自己。

世界让我活了半生

世界让我活了半生，但是世界并没有给我人生一半的精彩。

我在做着梦，要一直到这个世界的尽头吗？做梦吧！时不时还会把你拉入我的梦。为了意淫，还有手淫。这有什么不对？一个真正的天真的人，无知，无能，你还能给予他什么？世界总是向着善良的人们，总不能向着恶吧？为静静地恶来献媚？

世界说：“快来吧！孩子。”

我说：“㤙～嗯，不吗！”

世界又一次说：“快来吧！孩子。”

我说：“滚！”

世界还是那样善意的让我活着，也善良的总呼喊你的归来，我把它的呼唤当作了理所应当。

就这样，就这样一直，一直这样下去。就这样下去吧！也许会有这么一天，我与世界化敌为友。因为我是你的孩子！

补忆

大傻

2岁牙牙学语的时候，我带着我的女朋友，去水池边玩。不知道怎么回事，那时候就有泡妞诀窍了，我指着池子喊：“鱼，鱼。”便拿着我家乡添炕的推把，去捞了。小“女朋友”就在旁边看着，好像等待着我的意外收获。
当然了，既是我的女朋友，当然需要去认真对待她了。我很卖力的用推把去捞池子里的鱼，就这样太过卖力了，自己也钻进了池子。听后来，她长大了开玩笑给我说，我还在里面游泳。其实是这样的，她很聪明，便大喊起来，“娃娃！娃娃！” 不过确实是福大命大，单位上的职工，刚刚从市场上买瓜回来，一听到喊叫的声音，瓜一扔，二话没说，便来到池子旁边。北方人都是旱鸭子。不会水，就在旁边用刚刚我用过的推把捞，根本也是钩不到，无奈，只好有个子比较高的跳进了水中，把我捞了上来。有年轻人，也有老人在旁边把喊叫当作了指挥。一位老人喊叫着，让捞上来的那些年轻人把那个小尸体倒立的提着。他们听了，因为他们知道老人总说话是对的。他们身强力壮，他们不知，但他们知道敬重。他们提着我狂飙到医院的路上。他们好像还知道敬畏，对生命的敬畏，他们用她全身的力量，对待着死神的铁链。那个死去的小尸体动了，实则是他拉断了死神的铁链，但是他被吓到了，他停下了脚步，说：“我不行了，快换个人。”有一个身强力壮的他们把我提到了医院。
院长正在给新上任的医生露天教课，一看有情形不对，大喊到：“去抢救室！” 提我的他们也许不知道抢救室在什么地方，但是此时就是一个心领神会的天使，他当然是从天而降的万能的神。他没走一个曲线，直奔到了抢救室。
母亲被吓到了！
那一群战神被吓到了！
对于我来说，世界被吓到了！
此时，院长被吓到了。他不知道应该怎么去打开氧气瓶了，手脚无措了。旁边有经验的老神仙，他打开了氧气。
就此故事便结束了，我在这里忏悔。忏悔那些人都被我的无能而抛弃，被我的冷漠抛弃。
他们给我了生命，然后呢？浑身冷汗的我……
