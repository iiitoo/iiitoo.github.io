回家

       人最终总要回家，人的家在什么地方，所以人在路上。

       旅途的精彩，让人开始的时候兴奋不已，就忘记了回家的事。时而他会记起，但他总是回避或者借口，就像玩疯了的孩子。（动不动就说自己是成年人了，这只是我们想享受另外一些“成年人”说的长大后的奖励。不然兴奋的人怎么会去成长，难道成长真正地那么幸福吗？）精彩是时间的，而时间又是一个吝啬鬼，不会让你这么一直一直的精彩下去，又让你感到它的多余与厌倦。这时，人想回家了，玩的迷了路的人不知道家在哪个方向。所以，他还在刚才兴奋不已的旅途，丧气的慌乱的张望着美丽风景的最远处，寻找着八方的一个家。

神经病与正常人的故事

一秒一秒
不知时间的方向

神经病 
在那里等待。
坐标
灯塔
那个恶魔的眼睛，
吓得……
黑昼  、白夜

正常人
歧途
岔路
那双天使的臂膀，
护着……
12小时、12小时

发光的眼睛用他红红的光亮吸引
流血的翅膀用她红红的羽毛指正
神经病啊！
正常人啊！
你走向了疯狂
朝着那条歧途奔向恶魔。

网建随记

centos6.5笔记本wifi网络配置

参考：转载：http://u50.cn/blog/centos-6%E4%B8%AD%E9%85%8D%E7%BD%AE-broadcom-%E6%97%A0%E7%BA%BF%E7%BD%91%E5%8D%A1#comment-60259

有一台老的 dell 1400 笔记本，速度已经跟不上时代了，闲置下来。

拿它来做一个服务器，用于自己网站建设测试。

前面网卡驱动配置与参考文件不一样，再说，每个人电脑的网卡不同，就没有写出。无线上网，全靠下面引用的文章。

现在网卡配置好了，下一个任务是连接 WIFI 并在开机时启动

参考文档：http://www.pantz.org/software/wpa_supplicant/wirelesswpa2andlinux.html

 

1. 找 wifi 热点

sudo iwlist scan

2. 生成热点的配置（wpa2加密）

sudo wpa_passphrase <wifi热点名称> <wpa密码>

会生成这样格式的配置代码：

network={ 

        ssid="SSID_xxx" 

        #psk="your_pass_here" 

        psk=eb900177d2234487397b54f1fc1d6efc7f30e43b988ad21488ae3eb701a1g3b0 

}

将这段代码添加到 /etc/wpa_supplicant/wpa_supplicant.conf 文件中：

ctrl_interface=/var/run/wpa_supplicant 

ctrl_interface_group=wheel

network={ 

        ssid="SSID_xxx" 

        #psk="your_pass_here" 

        psk=eb900177d2234487397b54f1fc1d6efc7f30e43b988ad21488ae3eb701a1g3b0 

}

配置就完成了，下面添加启动代码

3. 在 /etc/rc.local 中添加相应的 wifi 启动代码：

pkill wpa_supplicant 

pkill dhclient

ifconfig wlan0 down 

ifconfig wlan0 up 

iwconfig wlan0 essid "SSID_xxx" 

wpa_supplicant -B -Dwext -iwlan0 -c/etc/wpa_supplicant/wpa_supplicant.conf

dhclient wlan0

现在重启系统，无线网络就会配置好了

补忆

叫魂

     在老家时有种叫魂的巫术。就是如果感觉自己的家人和自己有点精气神不足，就可以请巫术高强的巫师帮忙叫回自己的魂魄。有时也叫不回来，这主要看举行完这样的法事之后，这个人到底是不是有了精气神。我有幸做过一回这样的丢魂人。

    傍晚，刚刚从别处玩饿了，回家吃饭。一进家门，就感觉气氛不像往常。原来是家中有客了，一个不认识的客人和一个庄里很有威望的法师正在正房的炕上和我的家人聊天。家里的女人们就和往常有客人来家里一样忙碌着。因为饿或者总喜欢和妈妈、奶奶待在一起就直接去了厨房，她们一见我，眼神中带有神秘。我没多问，就随手拿了几样熟食吃去了。我知道，如果要解开这个神秘，就必须大着胆子到那群男人中打探消息。大着胆子战战兢兢来到正房，靠着门柱，等着别人的招呼。不出所料，来了一句，

    “快进来啊，靠着门跟要饭的一样。”父亲很严厉的这样说。

    转头从严肃的神情变为了笑脸又和这群男人聊天。我得到了招呼，心满意足的远远的坐到旁边听着他们的各种奇幻的故事。也听出了一点好像与我有关的事，就是我不小心把魂给丢了的事。心理总有许多说不出的感觉，可能是一种自责。我老丢东西，每次丢东西小则被训，大则被打。这次丢了个这么个东西，又不是东西的东西。总怕被打，又觉得冤枉。要说冤枉也没用，从前不小心丢的东西，也照样被训。但是此时，我并不怕训，因为有这么多陌生人为我撑腰。

   似懂非懂的论坛中听出人的灵魂的神奇。当然越是听神奇的事情，时间过得也是越快，终于到了实际操作的时候。今天，我是主角。一仗黄布把我围在炕角平躺，然后好像用另一种颜色的布从头到脚把我遮住，因为自己紧张忘了当时是用什么布蒙住我的；也因为紧张，我悄悄的一动不动的躺着。我只能用听觉去了解我周围的一切。法师一会敲打着房子里敲起来最响的东西，还叫着我反正一句没听懂的天语。各种吵闹后，平静好久。然后，听到院子里又是一片乱叫，时而还听到我家的男人们叫喊着我的名字。我知道这不是我的名字，这是我的魂的名字，就一动不动的躺着。也许是从前面的故事中得到了心得吧，此时没有我，只有我的魂。后来，走到我身旁不知何人，往我胸口放了一件东西，当时感觉有点难受。第二天一问，得知只是一个竹篮。立马就回到炕上，对法师从敬畏提升到了敬重。法师去掉胸上的东西后，一下感觉轻飘了好的，如升天。从法师的声音和敲打声慢慢变小，应该是收尾了吧。卸掉围帐，揭开蒙住脸的布。顿时，豁然开朗，如获新生。女人躲在偏房里偷看，男人对着高堂各种磕头作揖。主人和客人寒暄着要回自己家的话，好久才达到目的。

同事

太多的事，你们这些刚刚步入社会的人都太在乎了。
和每一个人都一样，他来到了这个没有人烟的戈壁滩上，对这样的一切只是排斥。奔土飞石的荒野，一眼望去便是天地之间的一线，有沙土，有狂风，有太阳，还有月亮。这一切当置身于此时，又在自己身上增添了恐惧。
有的人在恐惧中生活了40年、30年、20年、10年，5年、2年、12个月，还有今天刚刚来的他。有的人将自己置身于恐惧的一部分来吓唬自己和那些在恐惧中的人。人身上的美丽和丑陋都被放大，此时的他看到的总是丑多余美。吓的微低的头，黑眼珠总是一半，另一半藏在眼眶里。就这样，还用剩下的一半寻找自己身子的藏身之处。找不到时，是神给他了力量去勇敢的面对这样的恐惧。
嘶吼的机器只能让他显得刚刚像来到世间一样的无力，就连自己哇哇大哭的心灵被这样的声音所覆盖。关上房门，躲在窗后，偷偷用剩下勇气窥视这吼叫的东西。

碗

头顶飞沙，微微弯曲的身子横着想避开来袭的大风，提着一个刚刚吃完饭的铁皮盆去洗，走到洗衣房门口，不知从何处来的怒气无法释放，只有这手中的“碗”便向地上摔去，可能是没有达到想要的那种扁平效果，就冲到碗旁用钢包头的工鞋连踩数脚才心满意足。像一位疯狂艺术家一样完成了他的作品，走开了。

工作！他不想干的这个工作，还为了这样一个工作吃着方便面。哦！不对，方便面也许太贵，买上一个电饭锅熬一熬挂面，满足一下吃饭的过程吧，证明一下他还是一个人，有正常吃饭的需求。一日一到两餐，一日一日一日的慢慢走向迷茫的夜里。哪里有我们看到的绚丽星空，他没有这样的感受，因为他没有经历过我们所看到的美丽夜空。

摔完了碗，不用在吃破饭洗破碗了，如释重负。回到了房间低声下气的到别人旁边要上一根香烟，优雅的点着，又像一位伟大的思想家一样举起了低了半天的头，思考着。其实自己已飘到了天上。

“快去上班去！”

没有理，一下翻起身来。大步流星的狂奔在荒滩上，奔向那个不远的高塔。

礼上情在

躺在卧铺车厢中，如睡针毡啊！时时再说别人对自己的不好，而到了自己身上，总是对别人一般。这点道理我应该去懂。若连这些都不知道，怎么去做别的我想做的事情？
用自己好像是人的方式去炫耀，当把皮拔掉看到的是无知与冷酷，幼稚当然不能永远这么幼稚。若自己是一个艺术家，别人倒是可以原谅，但是自己又不是艺术家。也不能谈原谅之说，应该说的是以后的应对之策。修改，没有定型的修改。
我此时，在坐火车，我应该做什么？我明天要上班我应该做什么？未来我在计划，计划了，我该做什么？这样的一切只有一个勤快上说话，一个亲近上说话。人情关，是我要过的一大关，若过了。我才能考虑我该干什么。
